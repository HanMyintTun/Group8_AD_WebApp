Database and Web App Set Up:
----------------------------------------------------------------------------------
Please find in the Group8_AD_webapp Solution:
1. In App_Data Folder: aspnetdb.bak and SA46Team08ADProjectExtended.bak
2. If database has not been previously restored (Extended DB is same as for our Android App), 
    please restore the databases in SQL Server Management Studio to aspnetdb and SA46Team08ADProjectExtended 
3. Please publish the solution to a new Web Application Folder in Local IIS named Group8ADProject 
4. Navigate to localhost/Group8ADProject/Login.aspx via web browser (recommended: Google Chrome)
